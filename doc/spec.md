# プロジェクト仕様書: 多機能デジタル時計 PWA (React + MUI)

## 1. 概要
本プロジェクトは、常に表示しておくための「置き時計」としての利用を想定した、高機能なデジタル時計Webアプリケーションである。PWAとして構築し、オフライン環境でも時計機能が損なわれないように設計する。

## 2. 技術スタック
- **Framework**: React (Vite)
- **UI Library**: Material UI (MUI) v6
- **PWA**: vite-plugin-pwa (Workbox)
- **State Management**: React Context API / LocalStorage (設定保存用)
- **Icons**: Lucide React / MUI Icons
- **Weather API**: OpenWeatherMap API (or Similar)

## 3. 主要機能

### 3.1 デジタル時計ウィジェット
- 24時間/12時間表記の切り替え。
- 秒表示のオン/オフ設定。
- 現在の日付表示。

### 3.2 カレンダーウィジェット (オンライン時限定)
- 当月のカレンダー表示。
- オンライン復帰時に自動同期。

### 3.3 天気予報ウィジェット (オンライン時限定)
- 現在地の天候と気温を表示。
- 定期的なデータ更新。

### 3.4 オフライン対応 (PWA)
- Service Workerによる静的アセットのキャッシュ。
- オフライン時はカレンダーと天気を非表示（またはキャッシュデータを表示）し、「Offline Mode」のインジケーターを表示。

### 3.5 ウィジェット・マネージャー
- ユーザーが以下の表示項目をトグルスイッチで選択可能。
  - 時計 (必須)
  - カレンダー
  - 天気予報
- 設定はブラウザの LocalStorage に保存され、次回アクセス時も維持される。

## 4. レスポンシブ & デザイン設計

### 4.1 基本レイアウト
- **スマホ縦持ち (Portrait)**: ウィジェットを垂直にスタック。時計を最上部に配置。
- **スマホ横持ち (Landscape)**: 画面を左右に分割、または3カラム構成にする。時計を左側、その他を右側に配置し、フォントサイズを最大化する。

### 4.2 UI/UX
- **ダークモード推奨**: 置き時計としての眩しさを抑えるため、デフォルトをダークテーマに設定。
- **フルスクリーン対応**: ブラウザのUIを隠して表示する機能。

## 5. 画面遷移・構成
- `/`: メインダッシュボード
- `/settings`: 表示ウィジェット選択、テーマ設定、APIキー設定
